A subnet is a logical subdivision of your VPC's IP address range, used to organize and isolate resources.

Route Tables

A set of rules that directs network traffic from subnets.

Destination

The IP address range where traffic is going. E.g., `0.0.0.0/0` for all external traffic.

Target

The gateway or connection to send the traffic through. E.g., `Internet Gateway`.
