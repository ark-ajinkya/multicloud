``` Navigating the File System

Navigating the File System

Finding Your Way Around

üìÇ cd (Change Directory)

  • cd /path/to/dir: Go to specific directory
  • cd ..: Go up one directory
  • cd ~: Go to home directory

üìã ls (List Directory Contents)

  • ls: List current directory
  • ls -l: Long format (permissions, size, date)
  • ls -a: Show hidden files

üìç pwd (Print Working Directory)

Display the absolute path of your current location.

``` File Permissions and Ownership

Controlling Access to Your Files

``` Wildcards and Regular Expressions

Wildcards & Regex

Pattern Matching for Efficiency

‚≠ê Wildcards (Globbing)

  • *: Matches zero or more characters (ls *.txt)
  • ?: Matches exactly one character (ls photo?.jpg)
  • []: Matches any single char in brackets (ls [abc]file.txt)

üîç grep (Global Regex Print)

Search for patterns (regular expressions) within files.

grep -i "error" log.log (case-insensitive)

Find exactly what you need, fast!
